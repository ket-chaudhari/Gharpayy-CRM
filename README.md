# 🏠 Gharpayy CRM

A simple and efficient Customer Relationship Management (CRM) system designed to manage customers, services, and business operations in a structured and digital way.  
This project helps in organizing customer data, tracking services, and improving business workflow efficiency.

---

## 🚀 Features

- User Authentication (Login/Register)
- Customer Management (Add, Update, Delete customers)
- Service Tracking System
- Organized backend APIs
- Database integration for persistent storage
- Secure and scalable structure

---

## 🛠️ Tech Stack

- **Frontend:** HTML / CSS / JavaScript (or React if used)
- **Backend:** Node.js / Express.js
- **Database:** MongoDB / MySQL (as used in project)
- **Version Control:** Git & GitHub
